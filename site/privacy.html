AgentXL Privacy Policy

Last updated: March 12, 2026

What AgentXL is

AgentXL is a local-first Excel add-in that helps users turn source documents into traceable Excel workpapers. It is built by DeltaXY.

Data collection

AgentXL does not collect, store, or transmit any personal information.

  • No telemetry
  • No analytics
  • No cookies (beyond what Office.js requires)
  • No user accounts with DeltaXY
  • No data sent to DeltaXY servers

How AgentXL works

AgentXL consists of two components:

  • Excel add-in (taskpane UI) — runs inside Excel, loaded from GitHub Pages or localhost
  • Local server — runs on the user's own machine at localhost:3001, processes documents and communicates with the user's chosen AI model provider

Data that leaves your machine

When you ask AgentXL a question about your documents, the relevant content from those documents is sent to your chosen AI model provider (e.g., Anthropic, OpenAI, OpenRouter, Google) as part of the AI prompt. This is the only external communication.

DeltaXY does not operate the AI model. The user selects and authenticates with their own provider. Data is sent directly from the user's machine to the provider's API — DeltaXY is not an intermediary.

API credentials

API keys and OAuth tokens are stored locally on the user's machine (in ~/.pi/agent/auth.json or ~/.agentxl/auth.json). They are never transmitted to DeltaXY.

Local file access

AgentXL reads files from a local folder that the user explicitly selects. Files are processed locally on the user's machine. File contents are only sent to the AI provider when the user initiates a query.

Third-party services

AgentXL may communicate with the following third-party services, only when configured by the user:

  • AI model providers (Anthropic, OpenAI, OpenRouter, Google, GitHub Copilot) — for answering questions about documents
  • Mistral AI — for OCR processing of scanned PDFs (only if the user configures a Mistral API key)

Each provider has its own privacy policy. Users should review their chosen provider's terms.

Children's privacy

AgentXL is not directed at children under 13 and does not knowingly collect information from children.

Changes to this policy

We may update this policy from time to time. Changes will be posted on this page with an updated date.
