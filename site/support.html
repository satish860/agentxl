Common issues

Taskpane shows "Start AgentXL first"

The local AgentXL server is not running. Start it:

  • Windows release: Double-click AgentXL.vbs
  • npm: Run agentxl start

The taskpane will reconnect automatically once the server is available.

Certificate warning

If your browser shows a certificate warning when visiting https://localhost:3001, run:

npx office-addin-dev-certs install

Then restart Excel.

Add-in not visible in Excel

  1. Make sure the server is running
  2. Go to Insert → Get Add-ins → My Add-ins
  3. If not listed, re-upload the manifest or use the shared folder catalog method
  4. Close and reopen Excel

Port already in use

If port 3001 is occupied, use a different port:

agentxl start --port 3002

"No model available"

Run agentxl login to set up your AI provider credentials.
