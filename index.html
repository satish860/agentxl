AgentXL

Early Preview

AgentXL reads your local documents and maps them into Excel with source traceability. Point it at a folder of PDFs, bank statements, or audit files. Ask a question. Get the answer in your workbook with a citation back to the source file.

Before you start

  • Windows 10/11 with Excel (desktop)
  • An account with one of: ChatGPT, Claude, or GitHub Copilot
  • A folder of documents you want to work with (PDFs, text files, spreadsheets)

Setup

1

Download and install

Download AgentXL Installer

Windows installer · ~32 MB

Run the installer and follow the prompts. Installs Node.js runtime and downloads dependencies automatically.

Windows SmartScreen: If you see "Windows protected your PC", click More info → Run anyway. The scripts are not code-signed yet.
2

Sign in (one time only)

After install, open from your Start Menu or Desktop:

AgentXL Login

A browser tab opens. Sign in with your ChatGPT, Claude, or Copilot account. Once you see "success", close the tab.

3

Launch AgentXL

Open from your Start Menu or Desktop:

AgentXL

Excel opens with the AgentXL panel on the right side. If you don't see the panel, click the AgentXL button on the Home tab in Excel.

4

Link a document folder

In the AgentXL panel, paste the path to your local folder of evidence files. For example:

C:\Users\you\Documents\audit-files

AgentXL scans the folder and shows you how many files it found.

5

Ask a question

Type a question in the chat box. AgentXL reads your documents and gives you an answer with the source file referenced.

What to try

  • Link a folder with a few PDFs or text files and ask a factual question about the contents. "What is the total revenue in the annual report?"
  • Ask it to compare a document to what's in your Excel sheet. "Compare the bank statement to the cash column in this sheet."
  • Ask it to extract and map values from a document into specific cells. "Extract the cash balance from the bank statement and put it in cell B5."

Prefer npm?

If you have Node.js installed, you can skip the zip and install globally:

npm i -g agentxl  then  agentxl start

Windows says "protected your PC"

Click More info → Run anyway. The installer isn't code-signed yet (early preview).

AgentXL panel doesn't show in Excel

Close Excel completely and reopen it using the AgentXL shortcut from the Start Menu or Desktop. The panel should appear on the right side. If it still doesn't show, click the AgentXL button on the Home tab.

Already have AgentXL installed?

Download the new zip and extract it over your existing folder, or delete the old folder first. Your sign-in credentials (stored in your home directory) are preserved.

Something else went wrong?

Open an issue on GitHub or send us the error message.
