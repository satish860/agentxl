The problem

You open the folder. Dozens of PDFs, statements, exports. Then you open the workbook. And for the next few hours, you hunt.

Open a file. Search for the number. Copy it into Excel. Try to remember which document it came from. Repeat forty times. Then a reviewer asks "where did cell D14 come from?" and you do it all again.

What changes

Point AgentXL at the folder. Ask the question. Get the answer with the source file attached.

1

Choose the evidence folder

Select the local folder that contains your supporting documents — PDFs, statements, CSV exports, XLSX files, agreements. AgentXL scans and indexes them.

2

Ask a grounded question

"Extract the cash balance from the bank statement and map it to the cash workpaper." AgentXL searches the folder, reads the right files, and extracts the answer.

3

Review, verify, write to Excel

See which file the answer came from. Inspect the source reference. Then write it into the workbook — with a citation comment tracing back to the document.

"Where did this number come from?"

That is the question that matters most. Every output AgentXL produces traces back to a source file. You can verify the evidence before anything gets written to your workbook. No magic numbers. No unexplainable cells. Just documents → answers → traceable Excel.

Get started

Running in under two minutes. No account with us required.

Recommended

Windows download

Download the latest release, install or extract it, and launch AgentXL onboarding. It configures Office, opens Excel, and walks you through connecting your model provider and choosing your first folder.

Technical

npm install

Install globally with npm i -g agentxl, then run agentxl start. Works cross-platform. You bring your own model credentials — Claude, ChatGPT, Copilot, or an API key.
